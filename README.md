# Petrol Price Prediction — Linear Regression

Project notebook: Petrol_Price_Prediction_LinearRegression.ipynb

Overview
- Simple linear regression pipeline to explore and predict petrol prices using standard tabular features and EDA.

How to open
- Open the notebook in Jupyter or VS Code and run cells in order.

Requirements
- Python 3.8+ with common data-science packages: pandas, numpy, scikit-learn, matplotlib, seaborn, jupyter.
- (Optional) Create a virtual environment and install packages:

  pip install pandas numpy scikit-learn matplotlib seaborn jupyter

Data
- Place your dataset in a `data/` folder or the same directory as the notebook. The notebook expects the dataset path inside its cells — update paths there if needed.

Notes
- The notebook includes EDA, feature preparation, model training, evaluation, and basic visualizations.
